## 题干

定义函数：

$$
f(x) = \begin{cases}
\frac{1}{x} e^{-\frac{1}{x^2}} &, x \neq 0 \newline
0 &, x = 0
\end{cases}
$$

（1）证明 $ f(x) $ 在 $ x = 0 $ 可导.

（2）证明 $ f'(x) $ 在 $ \mathbb{R} $ 上连续.

（3）求 $ f(x) $ 单调区间、最大值点、最小值点.



### 证明

$ (1) $

$$
\lim\limits_{x \rightarrow 0} \frac{\frac{1}{x} e^{-\frac{1}{x^2}}}{x} = \lim\limits_{x \rightarrow 0} \frac{e^{-\frac{1}{x^2}}}{x^2} = \lim\limits_{t \rightarrow +\infty} \frac{t}{e^t} = 0, \quad t = \frac{1}{x^2} \to +\infty
$$

$ (2) $

在 $ x \neq 0 $ 处：

$$
f'(x) = \frac{e^{-\frac{1}{x^2}} (2 - x^2)}{x^4}
$$

而：

$$
\lim\limits_{x \rightarrow 0} f'(x) = \lim\limits_{t \rightarrow 0^+} \frac{e^{-\frac{1}{t}} (2 - t)}{t^2} = 2 \lim\limits_{t \to 0^+} \frac{e^{-\frac{1}{t}}}{t^2} = 2 \lim\limits_{k \rightarrow +\infty} \frac{k^2}{e^k} = 0 = f'(0), \quad t = x^2, \quad k = \frac{1}{t}
$$

$ (3) $

当 $ x \in [-\sqrt{2}, \sqrt{2}] $，函数递增； $ x < -\sqrt{2} $ 或 $ x > \sqrt{2} $ 时，函数递减.

分析两侧极限：

$$
\lim\limits_{x \to \infty} \frac{1}{x} e^{-\frac{1}{x^2}} = 0
$$

而 $ f(-\sqrt{2}) < 0, f(\sqrt{2}) > 0 $，故而其分别为最小值点、最大值点.

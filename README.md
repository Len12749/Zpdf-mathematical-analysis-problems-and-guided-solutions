# 《桃花依旧笑春风》习题册使用说明

这是一个用于整理数学分析习题与解析的仓库。

项目的目标不是单纯堆放答案，而是尽量把每一道题拆成可以复用的解题结构：题干、分析脚手架、正式证明。这样既方便回顾结论，也方便训练解题思路与书写习惯。

## 项目内容

仓库中的内容主要由按题号命名的文件组成：

- `n.md`：第 `n` 题的文字版解析
- `n.pdf`：第 `n` 题的排版导出版本

当前解析通常围绕以下结构展开：

1. `题干`
2. `脚手架` 或 `分析`
3. `证明`

其中“脚手架”部分用于提前说明这道题的切入点、关键观察、可调用的定理以及证明路线。它相当于正式解答之前的思维骨架，目的是让解析不仅有结果，也有方法。

## 文件约定

为了保持仓库整洁，建议继续沿用当前命名方式：

- 每道题使用统一编号
- 同一题的源文件和导出文件保持同名
- 一题对应一份独立解析，避免把多题混写在同一个文件里

推荐的单题模板如下：

```md
# 题号

## 题干

写清楚原题。

### 脚手架

- 这道题的核心观察是什么
- 准备使用哪些定理或变形
- 证明从哪里切入

### 证明

给出完整、可检查的推导过程。
```
